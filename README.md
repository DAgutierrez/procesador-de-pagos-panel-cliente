# Panel Web Cliente - Clubdelseguro

Aplicación React moderna para el panel de clientes de Clubdelseguro.

## Características

- Interfaz moderna y responsive
- Diseño limpio y minimalista
- Formulario de login con validación
- Integración con reCAPTCHA

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación se ejecutará en `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
